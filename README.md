# ずんだ通知読み上げもん

## PCに来る通知をずんだもんが読み上げるだけのアプリ

### 概要

PCに来る通知をずんだもんが読み上げてくれます

かわいいね

### 使い方

#### 初回インストール

1. `_install.bat`を実行する

#### 実行

1. `_main.bat`を実行する

### 機能

#### 有効/無効の切り替え

タスクトレイアイコンを右クリックして、有効または無効を押すと切り替わります。

無効になっていると、読み上げません。

#### 以上！！

漢に面倒な設定は不要！！
